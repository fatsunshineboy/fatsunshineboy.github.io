# 正则表达式

String str = "hello12af34";
Pattern p =  Pattern.compile("\\d{1,2}");
Matcher m = p.matcher(str);
while (m.find()){
    String s1 = m.group();
    System.out.println(s1);
}

java(?=8|11|17) 表示在java后面拼接8或者11或者17，获取只有Java java(?!8|11|17) 表示在java后面不拼接8或者11或者17，获取只有Java
